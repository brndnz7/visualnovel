## 🎯 **UTILISATION**

### Jouer
- **Clic** ou **Espace** : Avancer
- **H** : Historique
- **S** : Skip
- **A** : Auto

### Modifier le Scénario
**Fichier** : `src/data/story.json`
```json
{
  "speaker": "Mia",
  "text": "Salut !",
  "expression": "happy"
}
```
